## How To Reproduce

From `Snowball/anchor`:

```
npm run demo:devnet
```

Or directly:

```
ANCHOR_PROVIDER_URL=https://api.devnet.solana.com \
ANCHOR_WALLET=~/.config/solana/id.json \
npx ts-mocha -p ./tsconfig.json -t 1000000 scripts/devnet-demo-flow.ts
```

Each run generates fresh ephemeral keypairs and creates a new Campaign PDA, so the public keys, PDAs, and signatures above will differ on subsequent runs. The Program ID stays the same.

Provider wallet must hold at least 0.7 devnet SOL to seed the ephemeral creator, seller, and three buyers.

## Notes for Demo Video

This devnet flow proves the full escrow lifecycle:
create_campaign → join_campaign x3 → mark_shipped → confirm_delivery x2 → release_funds

No private keys are written to disk or printed. Ephemeral keypairs are discarded after the run.
